# Gushwork Frontend Assignment

This project is a **responsive web page implementation** built using **Vanilla HTML, CSS, and JavaScript**, following the design specifications provided in the Figma file. The goal of this assignment was to accurately reproduce the UI and implement interactive functionality without using any frameworks or external libraries.

## Live Demo

Live Website:  
https://gushwork-frontend-swarup.onrender.com/

## Figma Design Reference

Design specifications were followed from the provided Figma file:

https://www.figma.com/design/DOv07H7C2tA5UrVLhmfwfW/Gushwork-Assignment

The implementation aims to match the **layout, styling, spacing, and interactions as closely as possible** while ensuring responsiveness across devices.

## Features Implemented

### Responsive Design
- Fully responsive layout for:
  - Desktop
  - Tablet
  - Mobile
- Implemented using **CSS Flexbox and Grid**
- Media queries used to adapt layout and spacing across screen sizes

## Tech Stack

- **HTML5** – Structure and semantic markup  
- **CSS3** – Layout, responsiveness, animations  
- **JavaScript (Vanilla)** – Interactivity and dynamic behavior

No external frameworks or libraries were used.
